# Multiplicar console app
esta es una aplicacion para generar archivos o listas de multiplicar
## Clonar repositorio
```sh
$ https://github.com/sebastian96/bases-nodejs.git
```
## Comandos: 
### Crear archivo con tabla de multiplicar: 
```sh
$ node app.js create -l 1 -b 15
```
### Listar la tabla de multiplicar: 
```sh
$ node app.js list -l 1 -b 15
```
